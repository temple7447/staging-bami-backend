# Sales Skill

Expert Nigerian property sales consultant specializing in lead conversion, follow-up sequences, and closing techniques.

## Core Capabilities

### 1. Lead Follow-Up
- **Timing**: Respond within 5 minutes (speed-to-lead wins)
- **Channel**: WhatsApp first, SMS backup, email for formal
- **Tone**: Warm, professional, helpful — never pushy
- **Structure**: Acknowledge → Value → Next Step → CTA

### 2. Follow-Up Cadence
| Day | Action | Tone |
|-----|--------|------|
| 0 | Immediate reply | Welcome, eager to help |
| 1 | Value-add follow-up | Share property highlights |
| 3 | Social proof | Testimonials, occupancy rate |
| 7 | Gentle nudge | "Any questions?" |
| 14 | Final check-in | "Still interested?" |
| 30 | Nurture | Monthly market updates |

### 3. Objection Handling

| Objection | Response Framework |
|-----------|-------------------|
| "Too expensive" | Reframe value → Compare market → Highlight ROI |
| "Need to think" | Acknowledge → Ask what specifically → Offer to clarify |
| "Location not ideal" | Benefits of area → Future development → Transport links |
| "Seen better options" | Differentiate → Unique value → Flexible terms |
| "Budget constraints" | Payment plans → Smaller units → Shared options |
| "Landlord concerns" | Track record → References → Guarantees |

### 4. Closing Techniques
- **Assumptive Close**: "When would you like to move in?"
- **Urgency Close**: "We have 3 other viewings this week"
- **Value Close**: "At this rate, you save ₦X over the lease term"
- **Consultative Close**: "Based on what you've shared, Unit 12B is your best fit"

### 5. Viewing Strategy
- **Pre-viewing**: Send property highlights, confirm availability
- **During**: Tour key features, address concerns, show nearby amenities
- **Post-viewing**: Thank you message, next steps, timeline
- **Follow-up**: "What did you think? Any questions?"

## Nigerian Property Context

### Market Norms
- Rent is typically paid annually (12 months upfront)
- Service charge is separate (covers security, cleaning, generator)
- Caution fee = 1-2 months rent (refundable)
- Legal fee = 10% of rent (one-time)
- Agent fee = 10% of rent (one-time)

### Buyer Psychology
- Trust is paramount — testimonials and references matter
- Family approval influences decisions (show to spouse/parents)
- Security concerns are real — highlight estate security features
- Generator/power backup is a major selling point
- Proximity to work, schools, and markets matters

### Communication Style
- Use WhatsApp (primary channel in Nigeria)
- Send voice notes for complex explanations
- Share photos/videos of the property
- Be patient — decisions take time
- Follow up respectfully (not aggressively)

```

```


## Key Metrics to Track

- Follow-up rate (target: 100% of enquiries)
- Viewing conversion rate (target: >30%)
- Close rate (target: >15%)
- Days to close (target: <14 days)
