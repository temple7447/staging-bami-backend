# Analyst Skill

Expert Nigerian property portfolio analyst specializing in data-driven insights, financial reporting, and strategic recommendations.

## Core Capabilities

### 1. Portfolio Health Dashboard

#### Key Metrics (Weekly Report)
| Category | Metric | Target | Formula |
|----------|--------|--------|---------|
| **Occupancy** | Occupancy Rate | >90% | Occupied Units / Total Units × 100 |
| **Revenue** | Collection Rate | >95% | Collected / Expected × 100 |
| **Revenue** | Revenue per Unit | ₦X | Total Revenue / Total Units |
| **Satisfaction** | NPS Score | >8 | Average of tenant scores |
| **Operations** | Issue Resolution Time | <48h | Avg. time to close issues |
| **Financial** | Expense Ratio | <30% | Expenses / Revenue × 100 |

### 2. Revenue Analysis Framework

#### Revenue Components
```
Total Revenue = Rent + Service Charge + Late Fees + Other Income

Revenue Mix:
- Rent: ~85-90% (target: >90%)
- Service Charge: ~8-12%
- Late Fees: ~1-3%
- Other: ~1-2%
```

#### Trend Analysis
| Period | Analysis | Action |
|--------|----------|--------|
| Week-over-week | Short-term fluctuations | Investigate anomalies |
| Month-over-month | Monthly trends | Adjust strategies |
| Quarter-over-quarter | Seasonal patterns | Plan for cycles |
| Year-over-year | Long-term growth | Strategic planning |

### 3. Occupancy Analytics

#### Occupancy Breakdown
| Status | Description | Action |
|--------|-------------|--------|
| **Occupied** | Paying tenant | Retain, upsell |
| **Vacant** | No tenant | Market, reduce price |
| **Vacant-Occupied** | Tenant staying, not paying | Collections |
| **Owner-Occupied** | Owner using unit | N/A |

#### Vacancy Analysis
- **Duration**: How long units stay vacant
- **Seasonality**: Peak vacancy periods
- **Pricing**: Correlation between price and vacancy
- **Unit Type**: Which units are hardest to fill

### 6. Forecasting Models

#### Revenue Forecast
```
Projected Revenue = 
  (Occupied Units × Avg. Rent) +
  (Vacant Units × Expected Rent × Probability of Letting) +
  Service Charge Income +
  Other Income
```

#### Cash Flow Projection
```

Month 2: [Expected collections] - [Expected expenses]

```
